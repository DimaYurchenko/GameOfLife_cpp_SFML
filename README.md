# Conway's Game of Life

Implemented using c++ **SFML** library. 

To get started:
```
chmod +x build.sh
./build.sh
```
## Functionality

Simply **CLICK** on a cell to **activate/deactivate** it

Press **ENTER** to generate random cells

Press **SPACE** to **start/stop** simulation

Press **BACKSPACE** to **reset**
